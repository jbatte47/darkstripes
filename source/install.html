    Installation

    The only officially supported distribution channel for the Patterns libraries is NuGet. Head over to the list of Patterns libraries on NuGet for more information, or run the following commands from your NuGet Package Manager Console:

    To install Patterns, run the following command:

    PM> Install-Package Patterns

    To install Patterns.Autofac, run the following command:

    PM> Install-Package Patterns.Autofac

    To install Patterns.Testing, run the following command:

    PM> Install-Package Patterns.Testing

    To install Patterns.Testing.Autofac, run the following command:

    PM> Install-Package Patterns.Testing.Autofac
